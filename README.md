# Tick Distribution Modeling in Illinois

## Overview
This project analyzes tick abundance across Illinois counties using spatial modeling.

## Data
- Tick surveillance data
- Climate variables (WorldClim)
- Land cover (NLCD)
- Deer habitat suitability

## Methods
- Log transformation
- PCA
- Spatial BYM2 models (INLA)

## Outputs
- Maps of tick abundance
- Covariate panels
- Model diagnostics
